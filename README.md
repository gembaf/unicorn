## Installation

```sh
$ git clone https://github.com/gembaf/unicorn.git
$ cd unicorn
$ bundle install --path vendor/bundler -j4
$ bower install
$ bundle exec rake assets:precompile
$ bundle exec foreman start
```

## Requirements

- [Bower](https://github.com/bower/bower)
